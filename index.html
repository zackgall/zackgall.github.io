Projects

GitHub

Replay FX Social Wall

Connects Twitter Streaming API with Node server and MongoDB to capture pictures tweeted in real-time based on hashtags. An admin dashboard was developed to control settings and moderate the display to make AJAX calls to the database. The pictures/tweets are aggregated on a themed carousel template as part of a display at public events. Client can optionally auto-follow and re-tweet posts made under tracked hashtags.

Technologies: HTML, CSS, Javascript, JQuery, Bootstrap, AJAX, Twitter API, OAuth, MongoDB, Node.JS

GitHub

Hall Pass Generator

A single page web app that automates hall pass writing for high school blood drives. Entire application build on front-end with javascript and JQuery array of javascript objects that are placed into a template to display a list of hall passes. CSS was used to optimize for print by forcing page breaks and proper margins.

Technologies: HTML, CSS, Javascript, JQuery, JQuery UI

GitHub

Airport-on-Time

A one page web application that takes in user input to calculate what time they should leave to make it to the airport on time based on the TSA API and travel times from the user’s address using Google API.

Technologies: HTML, CSS, Javascript, Express, JSON, Node.JS, Google Maps API, MyTSA API
